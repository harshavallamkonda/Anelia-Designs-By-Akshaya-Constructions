# Anelia Designs Website

A modern, responsive website for Anelia Designs - Bangalore's premier construction and interior design company.

## Features

- **Clean Design**: White background with black text for optimal readability
- **Dynamic Package Display**: Packages are hidden by default and shown when "View Packages" buttons are clicked
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth animations and hover effects
- **Contact Form**: Integrated email contact form
- **WhatsApp Integration**: Floating WhatsApp button for instant communication

## How to Run Locally

1. **Download the project files** to your local machine

2. **Open the website** in any modern web browser:
   - Double-click on `index.html`, or
   - Right-click on `index.html` and select "Open with" → your preferred browser

3. **No installation required** - This is a static website that runs directly in the browser

## Key Functionality

### Package Display System
- Click "View Packages" buttons in the Services section
- Construction packages and Interior Design packages are shown separately
- Only one package section is visible at a time
- Packages are completely hidden by default (as requested)

### Interactive Dropdowns
- Each package has expandable sections (Structure, Kitchen, Bathroom, etc.)
- Click on any section to see detailed specifications
- Smooth animations for better user experience

### Contact Form
- Integrated contact form that opens the user's email client
- All form fields are validated
- Professional email template generation
